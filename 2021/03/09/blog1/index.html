准备工具

需要的工具有github,node.js,git,hexo;

1. 注册github

2. 安装git

输入git检测是否安装成功

3. 安装node.js

输入node -v和npm -v检查是否安装成功

4. 安装hexo

在D盘新建Blog文件夹,在该文件夹下输入npm install -g hexo-cli
安装完成后输入hexo init初始化,输入hexo g静态部署,输入hexo s本地查看
安装git部署插件npm install hexo-deployer-git --save
